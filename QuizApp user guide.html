1. Type 'cd code/QuizApp/uczlfr1' (' ' is not included) in Bitvise xterm.

2. Type 'node httpServer.js &' and then 'phonegap serve' (' ' is not included).

3. If you use a computer, connect UCL VPN first (if you do not use UCL website) and then in any browser type the link: http://developer.cege.ucl.ac.uk:31270/ If you use an Android phone, connect UCL VPN through AnyConnect (if you do not use UCL website), open PhoneGap app, type 'developer.cege.ucl.ac.uk:31270' (' ' is not included) and press 'Connect' button.

4. Now you can see an interface.

5. Press 'Track Location' button, then you will see your real-time position as a blue circle on map and you current coordinates on the sidebar.

6. When you are close to a certain place uploaded to database before, a popup will appear and on sidebar there is also a multiple-choice question.

7. Choose an answer from these four choices, tick in the circle. And then press 'Submit' button to insert your choice to database. If it is successfully uploaded, the words at the bottom will become 'Successfully submit'.

8. Check whether it is updated in database (The table is named UserAnswer).
